# MyWeatherApp

A simple Android weather application built with **Kotlin**, **Jetpack Compose**, and **Dependency Injection**.  
It fetches real-time weather data from a Weather API and displays **City**, **Temperature**, **Humidity**, and **Weather Description** in four clean cards.
